## 🚀 Quick Start
```bash
# create venv
python3 -m venv .venv
source .venv/bin/activate

# install
pip install -U pip wheel
pip install -r requirements.txt

# run
uvicorn gui.app:app --host 0.0.0.0 --port 8000 --reload
```
