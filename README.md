## ✨ Overview
Kydras Echo is a FastAPI-based utility (CLI/GUI) for quick local tooling.

## 🚀 Quick Start
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -U pip wheel
pip install -r requirements.txt
uvicorn gui.app:app --host 0.0.0.0 --port 8000 --reload
```
